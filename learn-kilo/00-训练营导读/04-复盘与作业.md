# 00 复盘与作业

## 复盘问题

请回答下面 5 个问题:

1. 这套课和普通源码阅读清单最大的区别是什么
2. “看过源码”和“能做二开”之间差了哪些能力
3. 你最容易掉回哪种旧学习方式
4. 你准备如何保证每节课都产出证据链
5. 你现在最期待通过这套课拿到什么能力

## 本节作业

提交 3 个结果物:

- 一张你的“当前理解版”仓库地图
- 一段不超过 200 字的学习目标说明
- 一份固定的学习笔记模板

要求:

- 三个结果物先按你自己的理解完成第一版
- 再用主教材和任务列表做一次修正

## 通过标准

只要你的回答里已经体现:

- 目标从“看源码”变成“做二开”
- 学习方式从“浏览”变成“先课程本体理解，再追链 + 输出”
- 后续每节课都要先课程讲透，再拿真实代码做证据

这一节就过关。
