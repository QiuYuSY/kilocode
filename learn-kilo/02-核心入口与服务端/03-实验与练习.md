## 实验 3：启动一次最小服务

在仓库根目录执行：

```powershell
bun run --cwd packages/opencode --conditions=browser src/index.ts serve --port 4096
```

记录：

- 终端具体打印了什么
- 哪句输出能证明服务真的启动了
- 你在 `serve.ts` 中找到了与这句输出对应的代码吗

如果你不方便长时间挂着服务，只要记录启动阶段的输出现象即可。
