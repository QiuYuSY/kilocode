# 02 复盘与作业

## 复盘问题

1. `index.ts` 最容易被低估的职责是什么
2. 为什么 `serve.ts` 里要处理优雅关闭
3. `server.ts` 的价值为什么不只是 route 聚合
4. `directory` 为什么能决定后面很多模块的边界
5. 为什么 SSE 是执行系统的一部分

## 本课作业

提交 3 个结果物:

- 启动主链图
- `server.ts` 的四层拆解图
- 一段 300 字以内的说明: “为什么 `serve` 是系统中枢”

要求:

- 先在不看源码的情况下完成第一稿
- 再回到源码补证据链

## 本课验收标准

- 你能把启动链讲通
- 你能解释上下文是在什么地方注入的
- 你不再把 SSE 看成附属能力
- 你已经能先讲系统，再拿代码验证
