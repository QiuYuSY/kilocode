# 03 复盘与作业

## 复盘问题

1. Session 和普通聊天记录最大的区别是什么
2. 为什么 Part 的存在对 UI 和调试都很重要
3. `SessionProcessor` 最容易被误解成什么
4. Agent 为什么不能只看 prompt
5. ToolRegistry 为什么不能写死

## 本课作业

提交 4 个结果物:

- 工具调用生命周期图
- 4 个 Agent 对比表
- Permission 执行流说明
- 一段“为什么 Session 不是聊天记录”的说明文

要求:

- 生命周期图和 Agent 对比表先按课程本体完成
- 再用源码修正和补证据

## 本课验收标准

- 你能先脱离源码解释会话执行系统，再用真实代码补证据
- 你能说清 Agent / Permission / Tool 的协作
- 你已经具备做小型行为改动的基础理解
