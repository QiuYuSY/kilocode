# 04 复盘与作业

## 复盘问题

1. 为什么不能把扩展端当系统主体
2. `ServerManager` 和 `KiloConnectionService` 的职责边界是什么
3. 为什么多个 webview 要共享同一个连接世界
4. Agent Manager 的复杂度主要来自哪里
5. 哪些问题明显不应该直接在 webview UI 里硬改

## 本课作业

提交 3 个结果物:

- 扩展接入链图
- `ServerManager` / `KiloConnectionService` 对比表
- 一段 “为什么 Agent Manager 不是另一套后端” 的说明

要求:

- 先根据课程本体完成接入链图
- 再回到代码验证共享连接和进程生命周期

## 本课验收标准

- 你能解释三层职责
- 你知道扩展是怎样接入核心运行时的
- 你已经知道扩展侧二开应先定位在哪一层
- 你已经能先讲结构，再找实现
