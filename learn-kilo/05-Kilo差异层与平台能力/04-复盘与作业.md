# 05 复盘与作业

## 复盘问题

1. Kilo 差异和品牌差异的区别是什么
2. 为什么 `config.ts` 会成为差异密集区
3. `kilo-gateway` 对整体系统意味着什么
4. 为什么 Telemetry 不只是外围能力
5. 为什么 `kilocode_change` 对 fork 项目很重要

## 本课作业

提交 3 个结果物:

- Kilo 差异落点清单
- 平台能力关系图
- 一段“为什么要集中管理 fork 差异”的说明

要求:

- 平台能力关系图先按课程理解完成
- 再用共享文件和 Kilo 专属包做差异验证

## 本课验收标准

- 你能区分共享能力和 Kilo 专属能力
- 你知道 Kilo 差异主要分布在哪里
- 你知道如何降低未来 merge 冲突
- 你已经能先讲平台差异，再找代码证据
