# 07 复盘与作业

## 复盘问题

1. 为什么验证要跟着改动类型走
2. 扩展端问题为什么要先抓连接主链
3. route 改动为什么容易漏 SDK
4. fork 项目里最重要的额外交付约束是什么
5. 你最容易漏掉哪一种检查

## 本课作业

提交 3 个结果物:

- 4 类改动的最小验证清单
- 一份扩展端连接问题的排查顺序
- 一页交付检查表

要求:

- 先凭课程理解写第一稿
- 再对照仓库真实命令和主链修正

## 本课验收标准

- 你已经能为改动写验证方案
- 你已经知道调试时优先抓哪条链
- 你已经知道这仓库的关键交付约束
- 你已经能把“课程里的交付方法”落回仓库真实流程
