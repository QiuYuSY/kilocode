# Learn Kilo Course Site

这是 `learn-kilo` 的本地课程入口页。

它现在不再只是“把课程标题列出来”，而是直接服务于新的目标:

让学习者先通过课程本体掌握 KiloCode 工程，再沿着真实主链、设计剖析和二开任务进入课程。

## 打开方式

直接在浏览器里打开:

- [index.html](./index.html)

如果你想验证相对链接和页面状态持久化，也可以在仓库根目录起一个静态服务后访问 `learn-kilo/course-site/`。

## 当前页面包含什么

- 两份主教材入口
- 一份实施任务列表入口
- 9 节课的训练路径
- 每节课的能力目标
- 每节课的设计解剖摘要
- 每节课的实验任务和验收点
- 对应的关键源码文件入口
- 本地学习进度记录

## 使用建议

推荐顺序:

1. 先看 `KiloCode架构总册.md`
2. 再看 `KiloCode二开手册.md`
3. 再看 `实施任务列表.md`
4. 再看 `00-训练营导读`
5. 再按顺序学 `01` 到 `07`
6. 最后做 `08-毕业项目`

## 说明

- 这是纯静态页面，不依赖构建工具
- 学习进度保存在浏览器 `localStorage`
- 课程入口页里的文档链接已改成相对路径，适合在不同本地目录下直接复用
- 真正的课程内容仍然在 `learn-kilo/` 各课包目录里
