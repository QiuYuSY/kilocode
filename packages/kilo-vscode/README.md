# Kilo Code

Open source AI coding agent for VS Code.

- [Install from VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=kilocode.Kilo-Code)
- [Home page](https://kilo.ai)
- [Quick-start video](https://youtu.be/pqGfYXgrhig)
- [Discord](https://kilo.ai/discord)

Kilo helps you build, ship, and iterate faster with an agent that can work directly in your editor.

## Core capabilities

- Generate code from natural language
- Check its own work
- Run terminal commands
- Automate the browser
- Get inline autocomplete as you type
- Use 500+ AI models including Gemini, Claude, GPT, Grok, GLM, and more
- Use Kilo-managed access or bring your own API keys
- Extend workflows with MCP servers
- Switch between Architect, Coder, and Debugger modes

## Get started

1. Install Kilo Code from the [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=kilocode.Kilo-Code)
2. Create an account to access 500+ AI models with transparent pricing
3. Open the Kilo sidebar and start coding with AI that adapts to your workflow

Watch the [quick-start video](https://youtu.be/pqGfYXgrhig) for a walkthrough.

## Developer setup

Run these commands from `packages/kilo-vscode/`:

```bash
bun run compile
bun run test:unit
bun run extension
```

### Snapshot builds

```bash
bun run snapshot:build
bun run snapshot:install
```

Snapshot versions include the current commit SHA and your git username.
